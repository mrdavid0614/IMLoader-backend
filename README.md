# ImLoader API

The API for an application to upload your images to the cloud and get the link of the image to share it.

## Technology Stack

**Development**
- Typescript
- Expressjs
- Yarn
- Multer

**Deployment**
- Railway

## Getting Started
```
# Install the required dependencies
$ yarn install;

# Run in development mode
$ yarn dev;

# Build app for production
$ yarn build;
```
